The point-classifier app

General aim:

The point-classifier app was designed to play around with k-means clustering algorithm. Lets see how!

Step one

The first step is to generate some random points. The user can define the total number, the standard deviation along the X and Y axis and the number of clusters.

points <- 500
sdX <- 2
sdY <- 2
gY <- 3
gX <- 4

x <- rnorm(points, 1, sdX) + 5*sample(c(1:gX),points,replace=TRUE)
y <- rnorm(points, 1, sdY) + 5*sample(c(1:gY),points,replace=TRUE)
plot(x,y)

Step two

After generating the points the server identifies the clusters.

df <- data.frame(x=x,
                 y=y)
kmeansObj <- kmeans(df,centers=gX*gY)
plot(x=x, y=y, col=kmeansObj$cluster)
points(kmeansObj$centers,pch=3,cex=3,lwd=3)
